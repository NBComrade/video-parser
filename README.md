# Video parser
Simple video parser, that can parse information form YouTube.

Requirements
------------

PHP >= 7.1

Installation
------------

You can get this project using git:

    git clone {github_link}
    composer install

Than run a php server :
```bash
$ cd public/
$ php -S localhost:8080
```
Go to the browser and open localhost:8080.
